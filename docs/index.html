Introdução

A ANSI C Data Structutres Library possui um conjunto de definições e funções de listas, pilhas e filas para uso destas estruturas de dados.

Até o presente momento, todas as estruturas existentes na biblioteca são estrutras estáticas, isto é, trabalham sobre vetores, sem nós ligados. Entretanto, estes vetores são alocados dinamicamente, o que permite tenham tamanhos definidos na chamada das funções de alocação.

Estas estruturas de dados também permitem armazenar qualquer tipo de dado, uma vez que trabalham com alocação dinâmica do vetor. Assim, estas estruturas são genéricas e podem armazenar qualquer tipo de dado, desde os tipos nativos da Linguagem C, até os criados por meio de structs, entre outros.

Sumário

  1. List - Lista estática com tamanho dinâmico;
  2. Queue - Fila estática com tamanho dinâmico;
  3. Stack - Pilha estática com tamanho dinâmico.

Instalação

Esta documentação está disponível no endereço https://iuricardoso.github.io/cdslib/.

A biblioteca está disponível para download no endereço https://github.com/iuricardoso/cdslib/. Ela é livre para ser utilizada em qualquer projeto.

Para utlizar a biblioteca, faça o download do arquivo cdslib.zip presente na pasta /zip/ da página do endereço acima descrito. Extraia o seu conteúdo, depois copie os arquivos extraídos para a pasta do projeto, e incluia-os na compilação.
