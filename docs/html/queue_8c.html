int queueAdd ( QUEUE  q,
const void *  element

Adiciona um elemento novo no fim da fila, retornando um valor lógico que indica se foi possível adicionar ou não.

Deve-se passar um ponteiro para o elemento a ser adicionado, sendo que este elemento deve ter o tamanho passado na função queueCreate. Este elemento será copiado para dentro da fila, não sendo referenciado por esta.

Deve haver espaço na fila para que seja possível adicionar o elemento. Caso exista, retorna o valor lógico verdadeiro (1). Caso não exista, esta função retorna o valor lógico falso (zero). A capacidade de armazenamento da fila é definida na sua criação, pela chamda da função queueCreate.

qA fila a ter o elemento adicionado.
elementO ponteiro para o elemento a ser adicionado.

O valor 1 caso tenha conseguido adicionar; 0 (zero) em caso contrário.

Carrega o elemento do início da fila no argumento element, retornando um valor lógico indicando se foi possível ou não carregar o elemento.

Caso a fila tenha elementos, a carga é bem sucedida, retornando o valor lógico verdadeiro (1); caso a fila esteja vazia, a carga não ocorre, retornando o valor lógico falso (zero).

ParÃ¢metros
qA fila a ter o elemento carregado.
elementO ponteiro para a variável que irá receber o elemento. Caso nenhum elemento seja carregado, a variável apontada pelo ponteiro permanece com o mesmo valor existente antes da execução da função.
Retorna
O valor lógico verdadeiro (1) caso tenha conseguido carregar um elemento; ou o valor lógico falso caso a fila esteja vazia antes da chamada desta função.
