◆ listAdd()

int listAdd ( LIST  l,
const void *  element 
)

Adiciona um elemento numa lista.

Parâmetros
lA lista a ter o elemento adicionado.
elementO ponteiro para o elemento a ser copiado para dentro da lista. O elemento é adicionado no final da lista.
Retorna
O tamanho atual da lista ou LIST_IS_FULL caso a lista já esteja cheia, indicando que não foi possível adicionar o elemento.

◆ listAddAtPosition()

int listAddAtPosition ( LIST  l,
const void *  element,
int  position 
)

Adiciona um elemento numa posição específica de uma lista.

  • Parâmetros
    lA lista a ter o elemento adicionado.
    elementO ponteiro para o elemento a ser copiado para dentro da lista. O elemento é adicionado no final da lista.
    positionA posição da lista ter o elemento adicionado.
    Retorna
    O tamanho atual da lista, ou um código de erro: LIST_IS_FULL ou INVALID_POSITION. O primeiro é retornado caso a lista já esteja cheia. o segundo, caso a posição seja ínvalida. Ambos os erros indicam que não foi possível adicionar o elemento.

◆ listRemove()

int listRemove ( LIST  l,
const void *  element 
)

Remove um elemento de uma lista.

O elemento passado como argumento é procurado na lista, e se encontrado pelo menos uma vez, sua primeira ocorrência é removida da lista.

Parâmetros
lA lista a ter o elemento removido.
elementO ponteiro do elemento a ter sua primeira ocorrência removida.
Retorna
O tamanho da lista após a remoção (um número maior ou igual a zero), ou a constante NOT_FOUND indicando que element não foi encontrado.
