◆ stackCapacity()

int stackCapacity ( STACK  s)

Recupera a capacidade máxima da pilha (a quantidade de elementos que a pilha pode comportar).

O capacidade da pilha é um valor inteiro c, em que c >= 0; indica a quantidade máxima de elementos que podem ser armazenados na pilha.

Parâmetros
sA pilha a ter sua capacidade recuperada.
Retorna
A capacidade da pilha.

◆ stackClear()

void stackClear ( STACK  s)

◆ stackPeekTop()

int stackPeekTop ( STACK  s,
void *  element 
)

Recupera o elemento do topo da pilha, sem desempilhá-lo.

Uma cópia do elemento é feita na variável apontada pelo ponteiro passado como parâmetro. Esta operação somente é realizada se a pilha não estiver vazia. Caso contrário, a variável passada como referência não sofre alteração alguma. O retorno da função indica se a operação foi bem sucedida ou não.

Parâmetros
sA pilha a ter o topo recuperado.
elementO ponteiro da variável que receberá uma cópia do elemento do topo da pilha.
Retorna
O valor lógico verdadeiro (1) caso tenha recuperado o valor; ou o valor lógico falso (zero) em caso contrário (pilha vazia).

◆ stackPop()

int stackPop ( STACK  s,
void *  element 
)

Desempilha o elemento do topo de uma pilha.

A função copia o elemento de dentro da pilha para a variável, retornando um valor lógico indicando se conseguiu desempilhar ou não. Não será possível realizar a operação caso a pilha já esteja vazia. Neste caso, a variável apontada pelo o ponteiro passado como parâmetro para receber o elemento desempilhado não sofrerá alteração.

Parâmetros
sA pilha que terá o elemento do topo desempilhado.
elementO ponteiro para a variável que receberá uma cópia do elemento desempilhado.
Retorna
O valor lógico verdadeiro (1) caso tenha conseguido desempilhar; ou o valor falso (zero) em caso contrário.

Empilha um elemento no topo de uma pilha.

A função copia o elemento para dentro da pilha, retornando um valor lógico indicando se conseguiu empilhar ou não. Não será possível realizar a operação caso a pilha já esteja cheia.

Parâmetros

elementO ponteiro para o elemento a ser empilhado.
Retorna
O valor lógico verdadeiro (1) caso tenha conseguido empilhar; ou o valor falso (zero) em caso contrário.

◆ stackSize()

int stackSize ( STACK  s)

Recupera o tamanho da pilha (a quantidade de elementos armazenados).

O tamanho da pilha é um valor inteiro n, em que 0 <= n <= c, sendo que c é a capacidade total da pilha; indica a quantidade máxima de elementos que podem ser armazenados na pilha.

Parâmetros
sA pilha a ter seu tamanho recuperado.
Retorna
int
